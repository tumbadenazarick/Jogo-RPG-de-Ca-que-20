# Infraestrutura DevOps Nexus Integrada

Este diretório contém as configurações para:
- **Prometheus**: Monitoramento
- **Circle CI**: CI/CD
- **ELK Stack**: Logs e Métricas
- **HashiCorp**: Terraform e Vault
- **SonarQube**: Qualidade de Código
- **Grafana**: Dashboards
- **Ansible**: Automação
- **Jenkins**: Automação CI
- **Kubernetes**: Orquestração
- **Docker**: Containerização
- **GitHub Actions**: Workflows
- **GitLab**: Gestão de Código
